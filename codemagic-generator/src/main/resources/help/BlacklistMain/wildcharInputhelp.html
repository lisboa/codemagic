The rule used to block forbidden words. This rule can contain the wildchars asterisk (*) and question (?). Use asterisk to match zero or more characters and question to matches zero ou one.

Example 1: The pattern *.domain.com blocks any text ended with domain.com , like page1.domain.com

Example 2: The pattern ab? matches ab and any 3 characters begins with ab, like abc, ab1, etc.
